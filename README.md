## Getting Started

No build step required. Open directly or serve locally:

```bash
# Using Python
cd mihir-portfolio
python -m http.server 8080

# Using Node.js
npx serve .
```

Then open `http://localhost:8080` in your browser.

## Sections

1. **Hero** — Particles, typing effect, orbiting tech icons, profile photo, code animation

3. **Skills** — Interactive cards grouped by category
